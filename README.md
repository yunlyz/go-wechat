# wxpay

全新的微信支付API v3 SDK for golang

SDK for golang 提供 V3 接口分类：
- [ ] 微信支付分
- [ ] 营销
  - [x] 代金券
  - [ ] 商家券
  - [ ] 小程序发券插件
- [ ] 电商收付通
- [ ] 其他
- [ ] 通用接口

## 安装
```shell
$ go get -u github.com/yunlyz/wxpay
```

## go module
```text
require github.com/yunlyz/wxpay latest
```
